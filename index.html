Bienvenidos todos a mi primer proyecto web.



Este es mi primer proyecto, solo creado con HTML y CSS.
En esta pagina web estoy creando un diseño responsive, Utilizando CSS GRID y FLEXBOX,
también he añadido dos @MEDIAQUERIES para el tamaño de escritorio y tableta.

Ya que el diseño original lo he empezado para teléfono móvil. Mi intención en este proyecto, es hacer algo básico y sencillo (tipo blog responsive).
Para empezar quiero moldear esta sencilla web con lo más básico, aunque a día de hoy podria meter JAVASCRIPT o podría trabajar también con REACTJS, BOOSTRAP etc,
Cosa que haré mas adelante.

Aquí te dejo los lenguajes y frameworks en los que estoy formandome, todo para desarrollo FRONT-END:
  • HTML5
  • CSS3
  • BOOTSTRAP
  • JAVASCRIPT
  • REACTJS
  • ANGULAR
  • JQUERY

Si queréis seguir conociendo más de mis proyectos y de los lenguajes de programación en los que me estoy formando
